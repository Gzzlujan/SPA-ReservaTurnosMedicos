# SPA, Webpack y tecnicas avanzadas de Js

Esta es una aplicacion de SPA construida con Javascript puro y Webpack

Este proyecto consiste en modificar la estructura de la aplicación web SPA de turnos médicos, creando el ambiente productivo con Babel y Webpack, emulando en el cliente una sesión de usuario a través del almacenamiento de credenciales de login en localStorage.

## Caracteristicas y tecnicas

- Enrutaminento con hash
- Modularidad, transpilacion y empaquetamiento
- Uso del almacenamiento con LocalStorage
- Control  del estado de la pagina ante el reinicio
